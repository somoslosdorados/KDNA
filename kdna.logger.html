kdna.logger.logger module

This is the logger program. It implements the “log” function. Logs the timestamp, the alert_level, the path from where the log is called and the message, in ~/.kdna/kdna.log

Notice: if you need to log anything, import this module and use the log function provided (logs its input).

Authors: Sarah THEOULLE, Théo TCHILINGUIRIAN

kdna.logger.logger.log(alert_level: str, msg: str)[source]

This function writes the given input to the logs file (log file is kdna/logs/kdna.log”).

Parameters:

  • alert_level – the importance or type of the logged message

  • msg (str) – the message to print and log

  • log_path (str) – the path to log to

Returns:

returns file.write output (that is, number of characters written)

Return type:

int
